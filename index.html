Crystal Collector Game

Instructions

  • A random number of crystal power is choosen to start the game.
  • Select a crystal to see how much crystal power it has.
  • You win if you get exactly the same crystal power asked for.
  • You lose if you get too much crystal power.
  • The game restarts whenever the player wins or loses.
  • When the game begins again, the player should see a new random number.
  • The random number shown at the start of the game should be between 19 - 120.
  • Each crystal should have a random hidden value between 1 - 12.
